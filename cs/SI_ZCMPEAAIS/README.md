# Silent Insiders: Zero-Click Memory Poisoning in Enterprise Agentic AI Systems

This repository holds the technical assets, architectural models, and generation metadata described in the research paper entitled **"Silent Insiders: Zero-Click Memory Poisoning in Enterprise Agentic AI Systems."**

The resources made available here support the threat models, attack scenarios (including Scenario 4: The Silent Monitor), and the proposed LLM07-X defensive system.

## 📂 Repository Structure
* **/figures**: Contains high-resolution manuscript figures categorized by format.
    * `/png`: Raster exports for web and general viewing.
    * `/svg`: Scalable vector graphics for high-quality publication rendering.
* **/prompts**: Documentation of the specific linguistic prompts used to generate the illustrative logic for the figures.

## 🎨 Visualization Methodology
